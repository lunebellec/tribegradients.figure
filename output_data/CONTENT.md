# Output Data

Generated by `uv run invoke run-visualize-gradients`.

## gradients/{subject}/

- `{movie}_average.png` — 4×4 surface maps (4 gradients × left/right × lateral/medial), averaged across all chunks for that movie.
- `{movie}_{chunk}.png` — same layout for each individual chunk.

PNG files are excluded from Git (see `.gitignore`).
