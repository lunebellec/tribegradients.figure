# Output Data

PNG files and PDFs are excluded from Git (see `.gitignore`).

## gradients/

Generated by `uv run invoke run-visualize-gradients`.

Directory structure: `subject-{id}/movie-{name}/gradient-{n}_chunk-{k}.png`

- `colorbar.png` — standalone colorbar for the gradient color scale
- `subject-{id}/movie-{name}/gradient-{n}_chunk-{k}.png` — brain surface map (lateral/medial × left/right hemispheres) for one subject, movie, gradient, and time chunk
- `subject-{id}/movie-{name}/gradient-{n}_chunk-average.png` — same layout averaged across all chunks for that movie

**Subjects:** 01, 02, 03, 05, tribe (group-level reference)  
**Movies:** bourne, figures, life, wolf  
**Gradients:** 1–4

## fig_tribe_gradients.svg / .png

Main figure, assembled manually from auto-generated assets in `gradients/` and `gradient_correlations/`.

- `fig_tribe_gradients.svg` — editable vector version
- `fig_tribe_gradients.png` — rasterized export for sharing

## gradient_correlations/

Generated by `uv run invoke run-gradient-correlations` (script: `analysis/gradient_correlations.py`).

- `gradient_{n}_correlations.pdf` — correlation analysis between human reference gradients and TRIBE subject/movie gradients, one PDF per gradient (n = 1–4)
